**1。**velocity db:
是的。NET 对象数据库，可以嵌入/分布和扩展到图形数据模型(VelocityGraph)数据库是快速和低内存消耗。VelocityDB 是一个性能极高、易于使用、可扩展、可嵌入和可分发的对象数据库系统。

**2。燕莎:**
这是由燕莎开发的面向物联网应用的时间序列数据库管理系统。它的服务器操作系统是 Window，支持 HTTP API。

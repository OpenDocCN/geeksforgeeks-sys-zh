# 传感器和执行器之间的差异

> 原文:[https://www . geesforgeks . org/传感器和执行器之间的差异/](https://www.geeksforgeeks.org/difference-between-sensor-and-actuator/)

**1。传感器:**
传感器是一种用于将物理事件或特征转换为电信号的设备。这是一种硬件设备，它从环境中获取输入，并通过转换将其提供给系统。
例如，温度计将温度作为物理特征，然后将其转换为系统的电信号。

![](img/6985541f1bc3b5f732e6b7f71777cddc.png)

**2。致动器:**
致动器是将电信号转换成物理事件或特征的装置。它从系统获取输入，并向环境提供输出。
例如，电机和加热器是一些常用的执行器。

![](img/07c208fafa2f3fba9bae36186a67412d.png)

**传感器和执行器的区别:**

<center>

| 传感器 | 执行机构 |
| --- | --- |
| 它将物理特征转换成电信号。 | 它把电信号转换成物理特征。 |
| 它接受来自环境的输入。 | 它从系统的输出调节单元获取输入。 |
| 它向系统的输入调节单元提供输出。 | 它向环境提供输出。 |
| 传感器产生电信号。 | 致动器产生热量或运动。 |
| 它被放置在系统的输入端口。 | 它被放置在系统的输出端口。 |
| 它用于测量物理量。 | 它用于测量连续和离散的过程参数。 |
| 它向系统提供有关环境的信息。 | 它接受命令来执行一项功能。 |
| 例子:将光能转化为电能的光电电池。 | 例如:步进电机，电能驱动电机。 |

</center>

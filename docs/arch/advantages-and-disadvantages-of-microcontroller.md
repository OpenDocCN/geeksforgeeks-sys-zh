# 微控制器的优缺点

> 原文:[https://www . geesforgeks . org/微控制器的优缺点/](https://www.geeksforgeeks.org/advantages-and-disadvantages-of-microcontroller/)

[微控制器](https://www.geeksforgeeks.org/introduction-to-8051-microcontroller/)是一个微电路(IC)，它将被编程以执行一组功能来调节一组电子设备。可编程是微控制器的独特之处。

**微控制器**可以是捕获输入、处理输入并生成输出的设备，支持捕获的知识。它也被称为 MC 或 MCU(微控制器单元)，可能是金属氧化物半导体芯片上的小型数字处理器。微控制器也被称为“专用计算机”。他们致力于执行特定的任务，这些任务被编程并存储在只读存储器中。它们配有板载存储器和输入/输出端口，无需构建具有独立外部随机存取存储器、只读存储器和外围芯片的电路。MC 的工作速度更低，需要更少的电力。微控制器的这些特性使其成为嵌入式应用的完美选择。

单板微控制器也被称为建立在一个计算机电路板上的开发板微控制器。存储器、时钟、输入输出电路等。它们普遍用于教育目的，以实现低成本的体验。

**微控制器分类:**

*   internal storage
*   structure
*   Bit configuration
*   instruction set

**微控制器的优势:**

*   The operation takes less time.
*   Easy to use, troubleshooting and system maintenance are simple and clear.
*   At the same time, many tasks are often performed, so manpower is usually saved.
*   The processor chip is extremely small and adaptive.
*   The cost and scale of the system are small.
*   The microcontroller is directly connected with additional memory, read-only memory and input and output ports.
*   Once the microcontrollers are programmed, they cannot be reprogrammed.
*   If the digital part does not exist, it will look like a microcomputer.
*   Easy to use, troubleshooting and system maintenance are simple and clear.

**微控制器的缺点:**

*   Generally used in micro devices.
*   Complex structure.
*   The microcontroller can't be directly connected to a better power supply device.
*   The number of executions is limited.
*   Because each microcontroller has no analog input/output, there are related problems.

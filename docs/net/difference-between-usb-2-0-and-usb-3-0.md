# USB 2.0 和 USB 3.0 的区别

> 原文:[https://www . geesforgeks . org/区别-USB-2-0-和-usb-3-0/](https://www.geeksforgeeks.org/difference-between-usb-2-0-and-usb-3-0/)

**USB(通用串行总线):**
A *通用串行总线(USB)* 是一种接口，可实现设备与主机控制器(如计算机系统)之间的通信。它连接数码相机、鼠标、键盘、扫描仪、打印机、媒体设备、闪存驱动器和外部硬盘驱动器等外围设备。它有多种用途，包括对电源的支持，这就是为什么通用串行总线取代了并行和串行端口等多种接口。
有不同类型的 USB 端口用于将外围设备连接到计算机。其中一些是:

1.  **USB 1.x:** 它是一种外部总线标准，能够以每秒 12 兆位的速度传输数据，最多可以支持 127 个外围设备。
2.  **USB 2.0:** 又名**高速 USB** 。它于 2000 年推出。它是 USB 1.1 的更新版本，提供了改进的功能和更好的速度。它能够提供每秒 480 兆比特的最大传输速度。然而，实际上它大约是 280 兆位/秒。
3.  **USB 3.0:** 又名**超高速 USB** ，于 2009 年 11 月首次上市。它是 USB 2.0 的改进版。它支持每秒 5gb 的数据传输速率，比 USB 2.0 提供的速度快得多。
4.  **USB 3.1:** It is also known as **SuperSpeed+**, and it was introduced in 2013\. USB 3.1 is the latest version of the USB protocol. It is almost two times faster than the USB 3.0 and can transfer the data at the rates of up to 10 Gigabits per second.

    **USB 2.0** 和 **USB 3.0** 是计算机系统和笔记本电脑中主要使用的设备。所以，在本文中，我们将讨论 USB 2.0 和 USB 3.0 之间的*区别。*

    **USB 2.0 和 USB 3.0 的区别:**

    | S.NO | USB 2.0 | USB 3.0 |
    | --- | --- | --- |
    | 1. | USB 2.0 的数据传输速率为 480 Mbps。 | 而 USB 3.0 的数据传输速率为 4800 Mbps。 |
    | 2. | USB 2.0 有四根线。 | 而 USB 3.0 有 9 根线。 |
    | 3. | USB 2.0 是单向通信。 | 而 USB 3.0 是双向通信。 |
    | 4. | USB 2.0 是一个成本较低的。 | 虽然它比 USB 2.0 贵。 |
    | 5. | USB 2.0 线缆长度可达五米。 | 而 USB 3.0 的线缆长度可达三米。 |
    | 6. | USB 2.0 提供 500 毫安的功率。 | 同时提供 900 毫安的功率。 |
    | 7. | USB 2.0 内部出现黑块。 | 而蓝色区块出现在 USB 3.0 内部。 |

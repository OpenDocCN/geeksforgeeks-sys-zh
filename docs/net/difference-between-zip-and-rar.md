# ZIP 和 RAR 的区别

> 原文:[https://www . geesforgeks . org/zip 和-rar 之间的差异/](https://www.geeksforgeeks.org/difference-between-zip-and-rar/)

**1。ZIP :**
是支持无损数据压缩和归档的文件格式。菲利普·卡兹是 1989 年这种文件格式的开发者。如今，ZIP 是一种广泛使用的格式，并得到了几个软件实用程序的支持，包括微软视窗和苹果操作系统 x 提供的内置 ZIP 支持。最棒的是，ZIP 文件可以用任何创建 ZIP 文件的程序打开。

Zip 文件是数据容器，其中包含一个或多个使用 ZIP 压缩的压缩格式的文件。好吧，ZIP 存档不仅仅能够压缩文件；他们只需点击几下鼠标，就可以加密文件(受密码保护)和分割档案。可以使用多种方法压缩多个文件，如 LZMA、WavPack、PPMd、BZIP2、DEFLATE 等。

**2。RAR :**
RAR 代表 Roshal Archive 压缩文件是一种专有的归档文件格式。它包含一个或多个文件或文件夹。

但是，与硬盘上的普通文件夹不同，RAR 文件需要第三方软件来打开和提取归档文件的内容。这是 WinRAR archiver 的一种本机文件格式，它以压缩形式存储多个文件，您只需打开其内容就可以访问这些文件。RAR 通常比默认支持的 ZIP 文件更擅长数据压缩。

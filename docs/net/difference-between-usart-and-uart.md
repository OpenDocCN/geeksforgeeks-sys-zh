# 【USART 和 UART 的区别

**USART :**
USART **(通用同步/异步接收器/发送器)**是一个附加在不同专用小工具上的协调电路。这些适合于改变序列结构中的并行信息。USART 的信息速度通常很高，因为它创建时钟数据或信息。USART 适合在输入顺序信息的交换请求中指导整个过程，并相应地创建信息的并行类型。在 USART 中，接收机不必知道发射机的波特率，因为它是从主机给出的信息线路和时钟信号中获得的。在 USART 中，发送外设产生一个时钟，接收外设接收该时钟，该时钟可以从数据流中检索，而不需要知道一段时间内的波特率。

**UART :**
UART **(通用异步接收/发送器)**是一款大规模集成小工具，有计划地处理顺序信息的传输。它(UART)位于以并行结构处理信息的框架和以不同寻常的顺序结构处理信息的小工具之间的中心。与 USART 相比，UART 提供了更低的数据传输速度。没有与信息相关的接近的时钟信号，因此接收者必须在开始收集之前知道发送者的波特率。在 UART 中，数据时钟由微控制器内部产生，并通过起始位转换与数据流同步。

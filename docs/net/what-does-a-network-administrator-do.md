# 网络管理员做什么

> 原文:[https://www . geesforgeks . org/网络管理员做什么/](https://www.geeksforgeeks.org/what-does-a-network-administrator-do/)

一个**网络管理员**的主要工作是保持组织网络的运行和更新。因此，任何拥有多台计算机的公司或组织都需要网络管理员(或管理员)来连接和协调不同的计算机或系统。

**网络管理员履行的职责:**

1.  **检测主机或路由器的接口卡故障:**
    A [路由器](https://www.geeksforgeeks.org/introduction-of-a-router/)或主机有多个接口与[互联网](https://www.geeksforgeeks.org/the-internet-and-the-web/)连接，我们连接的接口有可能出故障。因此，为了给用户留下深刻印象并吸引用户，网络管理人员可以通过使用一些网络管理工具来发现问题并通知用户。
2.  **主机监控:**
    网络管理员定期监控其网络内的所有主机，检查所有主机是否正常工作。
    它还可以通过阻止主机端的端口号来阻止特定主机使用特定服务(如 youtube 等)。

*   **在网络中添加或删除主机:**
    网络管理员可以在网络中添加或删除主机，还可以为主机配置子网掩码，这样主机将在网络中，但它将像在其他网络中一样工作。发送到该数据包的所有数据包首先到达网关，网关会将该数据包传输到该主机。*   **监控流量:**
    由于标准路由器只有物理层、数据链路层和网络层，所以只能进行路由(即制作路由表，借助 Link State [路由算法](https://www.geeksforgeeks.org/classification-of-routing-algorithms/)，决定路由)，为了路由，LSR 不会检测到某条路由上的拥塞(流量)。太多的数据包沿着同一条路径传输也会造成拥塞。*   **检测路由表中的快速变化:**
    当网络管理员对路由器配置不当时，路由器将无法正常工作，并且会非常快速地更改路由，最后发生故障。而从保存路由器下去管理员会发现自己的错误。*   **监控服务水平协议(SLA):**
    SLA 是定义具体绩效指标的合同。威瑞森和斯普林特只是众多向客户保证服务水平协议的网络提供商中的两家。这些服务级别协议包括服务可用性、延迟、吞吐量和停机通知要求。*   **安装、维修和更新软硬件:**
    如果网络的任何部分(如路由器、电缆或交换机等)工作不正常，网络管理员有责任将新的软件或硬件安装到网络中，以便更换或更新网络的该组件。*   **更新安全系统和安全授权:**
    入侵检测和提供额外的安全性是网络管理非常关键的职责。有几个系统试图入侵或关闭服务器或网络中的任何系统。例如，任何外部主机都试图通过发送许多 ping (echo) ICMP 数据包来关闭服务器，这导致性能下降。网络管理员可以很容易地检测到这种行为并阻止该 IP 地址。

**词法分析器:**

*   它是编译器的第一阶段，被称为**扫描仪**(它扫描程序)。
*   词法分析器将程序分成一些有意义的字符串，这些字符串被称为标记。

![](img/659b877e2419f498dfea68dfc04c27cd.png)

**令牌类型如下–**

1.  标识符
2.  关键字
3.  操作员
4.  常数
5.  特殊符号(@、$、#)

以上是在词法分析器中工作的关键组件——标记的术语。现在，借助示例，您将看到它是如何工作的。

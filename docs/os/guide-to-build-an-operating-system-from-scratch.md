[**操作系统**](https://www.geeksforgeeks.org/operating-systems/) **:**
一个**操作系统**充当计算机硬件和计算机用户之间的中介。操作系统的主要目的是为应用程序的有效执行提供一个环境。

操作系统管理设备、处理器和内存等资源和服务。流程管理、资源管理、存储管理和安全性是**操作系统**的一些重要功能。

## 下载

### Docker

```
docker pull apachecn0/geeksforgeeks-sys-zh
docker run -tid -p <port>:80 apachecn0/geeksforgeeks-sys-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g geeksforgeeks-sys-zh
geeksforgeeks-sys-zh <port>
# 访问 http://localhost:{port} 查看文档
```
